# Dataset Description

Dataset Source
The dataset was obtained from Kaggle:
https://www.kaggle.com/datasets/bhushandivekar/video-game-sales-and-industry-data-1980-2024?
Dataset Size
- Number of rows: 64016  
- Number of columns: 14  

Features Description
- title: Name of the game  
- console: Platform where the game was released  
- genre: Category of the game  
- publisher: Company that published the game  
- developer: Company that developed the game  
- critic_score: Review score given by critics (out of 10)  
- total_sales: Total global sales (in millions)  
- na_sales: Sales in North America  
- jp_sales: Sales in Japan  
- pal_sales: Sales in Europe and Africa  
- other_sales: Sales in other regions  
- release_date: Date the game was released  
- last_update: Date the data was last updated  

 Purpose of Using This Dataset
The purpose of using this dataset is to explore video game data and understand patterns in the gaming industry. The analysis helps identify trends over time, the most successful publishers, and how different factors such as critic scores may affect game sales.
